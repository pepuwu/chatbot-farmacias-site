# Chatbot Farmacias site

Landing estatica para usar como website publico en Meta Business / WhatsApp Manager.

## Deploy en Vercel

1. Importar el repositorio en Vercel.
2. Configurar `Root Directory` como `site`.
3. Framework preset: `Other`.
4. Build command: dejar vacio.
5. Output directory: dejar vacio.
6. Deploy.

Despues de publicar, usar la URL final en Meta:

```text
Business profile -> Website
```

Antes de pedir revision, reemplazar `contacto@chatbotfarmacias.com` por un email real si ese dominio todavia no existe.
