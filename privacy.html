Politica de privacidad

Chatbot Farmacias es operado por Esteban Pedro. Esta politica describe de forma general como se tratan los datos vinculados al uso del sistema.

Datos tratados

El sistema puede procesar datos de contacto, mensajes enviados por usuarios, informacion de farmacias, administradores y registros operativos necesarios para prestar el servicio.

Finalidad

Los datos se utilizan para responder consultas, gestionar pedidos, derivar conversaciones a administradores autorizados, mantener historial operativo y mejorar la calidad del servicio.

Conservacion y seguridad

La informacion se conserva durante el tiempo necesario para la operacion del servicio y se protege mediante controles razonables de acceso y almacenamiento.
